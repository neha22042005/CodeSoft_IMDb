# 🎬 IMDb Movie Rating Prediction

This project predicts movie ratings using Machine Learning based on features such as genre, director, and actors. The goal is to analyze how different factors influence a movie’s rating and build a model that can estimate ratings for new movies.

## 📊 Features Used

* Genre
* Director
* Actor

## 🧠 Model Used

* Linear Regression

## 🎯 Objective

To develop a predictive model that estimates movie ratings based on important attributes and understand the impact of different features on ratings.

## 🛠️ Technologies Used

* Python
* Pandas
* Scikit-learn

## 📌 Conclusion

The model provides a basic understanding of how movie attributes affect ratings. Although simple, it demonstrates the use of regression techniques in real-world datasets and can be further improved with more features and advanced models.
